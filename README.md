
# 🌍 AI Language Translation Agent

An AI-powered multi-language translation application built using Streamlit and Google Gemini.

---

## 🛠️ Tech Stack

- Frontend: Streamlit
- AI Model: Google Gemini (via google-genai SDK)
- File Processing: python-docx, PyPDF2
- Environment Management: python-dotenv

---

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/ai-translation-agent.git
cd ai-translation-agent
````

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Set up environment variables

Create a `.env` file in the root directory:

```env
GEMINI_API_KEY=your_api_key_here
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

---

## 📂 Supported File Types

* `.txt`
* `.docx`
* `.pdf`

---

## 🧠 How It Works

1. User inputs text or uploads a file
2. Gemini processes and translates the content
3. Output is enhanced based on selected tone
4. Optional improvements and speech playback available
